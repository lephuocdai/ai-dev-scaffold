# Deep Research

Structured research investigations conducted before introducing new technologies or approaches.

## Structure

```
docs/research/
  prompts/       # Research questions and scope
  results/       # Findings and recommendations
```

Research is stored as prompt/result pairs for reproducibility.

## When to Research

- Before choosing a new technology or framework
- Before making architectural decisions with long-term impact
- When evaluating competing approaches
- When entering an unfamiliar domain
